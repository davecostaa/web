Descobrindo o nome do Pod:

kubectl get pod -n <namespace> | kubectl get pod

Para acessar o Pod:

kubectl exec --stdin --tty <nome do pod> -n <namespace> -- /bin/bash

Também é possível entrar com uma sessão do shell:

kubectl exec --stdin --tty <nome do pod> -n <namespace> -- /bin/sh

Para rodar o Pod com uma imagem específica e acessá-lo:

kubectl run -i --tty <nome do pod> --image=<image> -- /bin/bash

E para excluir o Pod:

kubectl delete pod <nome do pod>
